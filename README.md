# Kicad Gamepad Stuff

Contains modified or newly created lib files and footprints for use in Kicad.
This collection of lib files and footprints were used while creating PCBs for Raspberry Pi gamepad projects.

## List of Components:

- ABXY_Button_Pads
- ADS1015
- Analog_Joystick
- Button_Pad
- Circular_Pad
- Hole
- LM339D_ADC
- M_USB
- Mounting_Hole
- Reverse_Tact_Button
- Reverse_Tact_Button_5.25mm
- Reverse_Tact_Button_5.75mm
- Reverse_Tact_Button_6.1mm
- SMD_0805_Resistor
- SMD_1206_Resistor
- SMD_2010_Resistor
- SW_3.5mm_Jack
- Select_Start_Button_Pads
- Small_Button_Pad
- Stereo_Vol_Wheel
- Tact_Button

- Through_Hole_R
- UDLR_Button_Pads
- USB_A
- Various Pins / Holes
